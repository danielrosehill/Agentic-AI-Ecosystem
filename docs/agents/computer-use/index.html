    Computer Use¶

    Agents / Computer Use

    Agents that drive a full computer via screen, keyboard, and mouse to accomplish user tasks.

    Agents whose action space is general desktop/OS control — screenshots, clicks, keystrokes — rather than narrow APIs or browsers alone. Includes local harnesses that wrap vision-language models to operate a machine. Browser-only automation and narrow RPA belong elsewhere.

    Subcategories¶

    • Local Computer-Use Harnesses (3 proj) — Harnesses that run computer-use agents locally on the user's own machine.
